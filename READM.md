# C type - python project

### Description
#### Name: Countdown Timer ID:23 Difficulty: C Propose: ALM

    Să se creeze o aplicație cu o interfață grafică ce reprezintă un temporizator. Trebuie să
    se poată seta un anumit număr de secunde, minute și ore. De asemenea, când timpul
    setat s-a scurs, să se notifice utilizatorul atât vizual, printr-un mesaj, cât și printr-un
    semnal sonor.
    INPUT: timer.py
    OUTPUT: interfața grafică cu timer-ul



### Flow:
    1. Se deschide fereastra cu timer-ul
    2. Se seteaza timpul
    3. Se apasa butonul de start
    4. Se asteapta pana cand timpul expira
    5. Se afiseaza mesajul si se aude semnalul sonor
    6. Se inchide fereastra
